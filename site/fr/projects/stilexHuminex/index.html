Stilex Huminex - Site Web

TL;DR

Stilex Huminex est un site réalisé à la main entièrement en HTML CSS par une équipe de 5 individus.

Qu'avons-nous fait?

Le projet HTML CSS du premier semestre à l'IUT nous demandait de créer un site internet faisant la promotion d'un produit fictif. Nous avons choisi la pile corticale de la série de livre et de films "Altered Carbon". Nous avons donc utilisé nos connaissances en HTML et en CSS afin de faire la promotion du produit de la façon la plus esthétique qui soit. J'ai demandé à ma soeur de nous fournir un code couleur, et nous avons réalisé le thème du site avec. J'ai donc créé la majorité du thème du site et rédigé la grande majorité du texte afin de rendre l'histoire de l'entreprise fictive Stilex-Huminex d'autant plus intriguante.

Qu'ai-je appris de ce projet

Ce projet HTML CSS m'a permis d'approfondir mes compétences dans ces domaines, notamment l'utilisation de flexbox et des différentes balises. Le site est également totalement responsive, et utilise de nombreuses fonctionnalités. Je me suis également servi de ces compétences afin d'utiliser la template pour créer ce portfolio.
